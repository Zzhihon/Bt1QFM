# Bt1QFM - 个人音乐电台

> 一个现代化的个人音乐流媒体平台，支持本地音乐管理和网易云音乐搜索播放
> 本地音乐管理主要是为了给那些有光驱的人，把抓轨后的音频文件批量上传和管理

## ✨ 功能特性

### 🎵 音乐播放
- **多格式支持**: MP3、WAV、FLAC、AAC、M4A
- **HLS流媒体**: 高质量音频流播放
- **播放控制**: 播放/暂停、上一首/下一首、音量控制
- **播放模式**: 顺序播放、单曲循环、列表循环、随机播放
- **进度控制**: 拖拽进度条调整播放位置

### 📚 音乐库管理
- **本地上传**: 支持单曲和批量上传
- **专辑管理**: 创建、编辑、删除专辑
- **元数据支持**: 歌曲标题、艺术家、专辑、封面
- **封面展示**: 自动提取或手动上传专辑封面

### 🔍 第三方音乐
- **网易云音乐集成**: 搜索和播放网易云音乐
- **智能搜索**: 通过 `/netease [歌曲名称]` 命令搜索
- **在线播放**: 无需下载即可播放网易云音乐
- **播放列表**: 将在线音乐加入播放列表

### 🤖 AI 音乐助手
- **聊天界面**: Discord风格的聊天界面
- **音乐搜索**: 通过对话搜索音乐
- **智能推荐**: 基于用户偏好推荐音乐
- **多频道支持**: 模拟多频道音乐体验

### 🎨 界面设计
- **主题切换**: 4种内置主题（赛博朋克、极简主义、暗夜模式、复古风格）
- **响应式设计**: 支持桌面端和移动端
- **现代UI**: 基于Tailwind CSS的现代化界面
- **动画效果**: 流畅的过渡动画和交互效果

### 👤 用户系统
- **用户注册/登录**: 安全的用户认证系统
- **个人档案**: 用户信息管理
- **权限管理**: 基于JWT的访问控制

## 🛠 技术栈

### 前端
- **React 18** - 现代化React框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 原子化CSS框架
- **React Router** - 路由管理
- **HLS.js** - 视频/音频流播放
- **Lucide React** - 现代图标库

### 状态管理
- **Context API** - React状态管理
- **Local Storage** - 本地数据持久化

### 音频处理
- **Web Audio API** - 音频播放控制
- **HLS (HTTP Live Streaming)** - 流媒体播放
- **音频格式支持** - 多种音频格式解码

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm 或 yarn
- 现代浏览器（支持Web Audio API）

### 安装依赖
```bash
cd web/ui
npm install
```

### 启动开发服务器
```bash
npm start
```

### 构建生产版本
```bash
npm run build
```

## 🎮 使用说明

### 播放音乐
1. 在音乐库中点击任意歌曲开始播放
2. 使用底部播放控制栏控制播放
3. 点击播放列表按钮查看当前队列

### 上传音乐
1. 点击"上传音乐"按钮
2. 选择音频文件（支持拖拽上传）
3. 填写歌曲信息和封面
4. 确认上传

### 搜索网易云音乐
1. 进入Bot页面
2. 输入 `/netease 歌曲名称`
3. 点击搜索结果播放或添加到播放列表

### 创建专辑
1. 进入专辑管理页面
2. 点击"创建新专辑"
3. 填写专辑信息和上传封面
4. 添加歌曲到专辑
